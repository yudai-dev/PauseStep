# 学習着手支援 Chrome拡張機能 v0.3.3

YouTube・X・Instagramを新しいタブで開いたときに、10秒間の介入を表示するChrome拡張機能です。

## 今回の修正

- `handleShouldIntervene is not defined` エラーを修正しました。
- 抜けていた介入判定処理を `service-worker.js` に復元しました。
- 同じSNSタブ内では介入を一度だけ表示する仕様を維持しています。
- 成功画面の「流石です！」と右上の課題導線は変更していません。

## 更新方法

1. ZIPを解凍します。
2. Chromeで `chrome://extensions/` を開きます。
3. 現在の「学習着手支援」を削除します。
4. 「パッケージ化されていない拡張機能を読み込む」から、解凍した新しいフォルダを選びます。
5. 課題URLが残っていない場合は、拡張機能アイコンから再設定します。
